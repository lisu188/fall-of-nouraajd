## controls
arrows - move
space - skip turn / close window
i - inventory
j - quest log
c - character sheet
s - save
tab - open/close python console
## running
### ubuntu
<pre>
./configure.sh
cd cmake-build-release
make _game_unity
python3 play.py
</pre>
